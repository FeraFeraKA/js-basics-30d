# Репозиторий по обучению прикладному JS в течение 30 дней

**Зачем нужно: устранить базовые пробелы в знаниях и подучить JS к Junior-уровню**

## Как запустить любой Day:

1. Установить расширение Live Server в VS Code.
2. Кликнуть ПКМ по index.html и нажать "Open with Live Server".

## Мини-Проект-1: Каталог с использованием FakeStoreAPI

Deploy: https://feraferaka.github.io/js-basics-30d/

## Реализовано:

- Загрузка товаров через API
- Поиск по имени с debounce
- Фильтр категорий
- Кнопка Reset
- Постепенный показ страницы (Показать ещё)
- Возможность добавлять товар в избранное с сохранением после перезагрузки
- Модальное окно карточек
- Статус загрузки
- Базовый a11y

## Технологии

HTML, CSS, JS, ES Modules

## Как запустить локально:

1. Клонировать ветку deploy-catalog в локальный git
2. См. "Как запустить любой Day".
